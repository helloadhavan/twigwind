Overview

This demo docs site is built with the Twigwind runtime generator + base utilities. It shows: colors, spacing, sizes, layout helpers, responsive prefixes (sm:, md:, lg:), hover: and custom arbitrary values (color-#ff6600, bg-rgb(...)).

Spacing

Use m-*, p-* and axis shortcuts mx-*, py-* etc. Values accept px/rem/em/% (e.g. p-12px).

p-20px m-10px
px-30px py-10px

<div class="p-20px m-10px">Box</div>
<div class="px-30px py-10px">Box</div>

Sizes

size-sm
size-md
size-lg

<div class="size-sm">Small</div>
<div class="size-md">Medium</div>

Grid

Twigwind grid syntax: grid:cols,rows,gap — e.g. grid:3,1,12px

2
3
4

<div class="grid:2,2,12px">
  <div>1</div>
  <div>2</div>
  <div>3</div>
  <div>4</div>
</div>

Flex

</div>

<div class="flex:row-center-center">...</div>
